# EnumAR - Web Augmented Reality with Three.js

Aplikasi Web Augmented Reality (AR) interaktif yang menempatkan model 3D menggunakan **Three.js** dan browser **WebXR Device API**

---
## Tips & Panduan di Mode AR
* **Deteksi Permukaan**: Saat masuk ke mode AR, gerakkan HP Anda memutar secara perlahan ke arah lantai atau meja yang terang. Gerakan ini membantu sensor AR mendeteksi kedalaman permukaan.
* **Indikator**: Ketika permukaan terdeteksi, lingkaran ungu akan muncul di permukaan tersebut.
* **Menaruh Model**: Ketuk layar sekali di posisi lingkaran untuk menempatkan model 3D logo.
* **Menyesuaikan**: Geser slider **Ukuran** dan **Rotasi** di panel bawah untuk memodifikasi model yang sudah ditempatkan.
